本篇文章將說明 Android 的 Booting Process.
<!--more-->

基本上 Android 的 Booting Process 可以分成七個階段:
1. Power on and boot rom code execution
2. The bootloader
3. The linux kernel
4. The init process
5. Zygote and Dalvik
6. The system server
7. Boot complete
